# Replit Build Prompt: Upgrade Partner Portal Internal Order Emails for NetSuite Quoting Workflow

We need to upgrade the Partner Portal internal order notification emails so they function like a complete project manager intake packet for A3 Visual. The goal is for the internal email to contain everything a PM, salesperson, or estimator needs to manually enter the order into NetSuite and generate a quote without hunting through the admin dashboard.

The current internal emails are too basic. They show only order reference, event, item names, and quantity. We need a full operational order packet.

## Core Objective

Build a detailed internal order email template and backend data mapping system that includes:

1. Full customer information
2. Partner information
3. NetSuite customer number
4. Salesperson associated with the partner or order
5. Contact details
6. Billing information
7. Shipping or delivery location
8. Event information
9. Order timeline
10. Package breakdown
11. Every item inside each package
12. Product dimensions
13. Hardware quantities
14. Rental inventory logic
15. Uploaded or associated reference images
16. Suggested vendor items that match the order
17. Admin entered notes and quote preparation notes

Think like a project manager who needs to enter this order manually into NetSuite to generate a quote for the customer.

# Part 4: Rental Inventory Logic

Some partner portals may include inventory or rentable customer owned stock.

The internal email must identify whether any ordered item should be:

1. Produced new by A3
2. Rented from partner owned customer stock
3. Pulled from A3 stock
4. Ordered from a third party vendor
5. Confirmed manually

## Rental Fields

For each relevant item, include:

- Rental Required: Yes or No
- Rental Source:
  - Customer Owned Stock
  - Partner Owned Stock
  - A3 Owned Stock
  - Third Party Rental
  - Unknown
- Available Inventory Quantity
- Quantity Requested
- Quantity Remaining After Order
- Inventory Location
- Inventory Contact
- Return Required: Yes or No
- Return Date
- Damage Liability Notes
- Replacement Cost, if available
- Rental Notes

## Inventory Warning Logic

Add automatic warnings in the internal email:

- If requested quantity exceeds available stock, show:
  `Inventory Warning: Requested quantity exceeds available stock. PM must review before quoting.`

- If rental source is unknown, show:
  `Rental Source Requires Review.`

- If item has both print production and rental hardware, separate them clearly:
  - Printed graphic production
  - Hardware rental or pull
  - Install labor
